Beginner Projects

Simple Scarf

This project is a simple scarf any beginner can make with a singular stitch pattern, and some patience! It's great for learning to change rows, and build familiarity with the movements of crochet. Find the video here.

Simple Bow

This project is a bow made of two rectangular pieces! This makes it very easy to make, and has a lot of room for easy customizability. Find the video here.

Circular Coaster

This project is a simple coaster that uses mostly double crochet stitches. This makes it great for picking up the fundamentals, as well as the basics of granny circles, which are used in a variety of projects! Find the video here.
